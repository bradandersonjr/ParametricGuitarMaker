# Templates Directory

Place your Fusion fretboard templates here:

- `fretboard_inch.f3d` — inch-based template
- `fretboard_mm.f3d` — millimeter-based template

## Requirements
- Both files must use the same user parameter names.
- Each should include a `TEMPLATE_VERSION` user parameter (e.g. "1.0.0").
- Save with timeline marker at start (no geometry visible on initial import).
- Fretboard should be its own component (not loose geometry in the root).
